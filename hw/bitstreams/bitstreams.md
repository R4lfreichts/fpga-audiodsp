# Bitstreams

Dieser Ordner enthält lauffähige Bitstreams für schnelle Zwischenstände zum programmieren des FPGAs.

## Aktueller Stand
- `i2s_sine_cora_z7_v0_1.bit`
  - Board: Digilent Cora Z7-07S
  - Funktion: I2S Sinusgenerator
  - Vivado: 2023.2

## Laden auf das Board
1. Vivado öffnen
2. Hardware Manager starten
3. Open Target → Auto Connect
4. Rechtsklick auf Device → Program Device
5. `.bit` Datei auswählen
